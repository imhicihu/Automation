### Anatomy of Macros 
#### Neofinder
```
{+KC_LSFT}{+KC_N}{-KC_LSFT}{-KC_N}
```
#### Zoom in in mac
```
{+KC_LGUI}{+KC_RBRC}{-KC_LGUI}
```
#### Zoom out in mac
```
{+KC_LGUI}{+KC_SLSH}{-KC_LGUI}
```
#### Paste
```
{+KC_LGUI}{+KC_V}{-KC_LGUI}{-KC_V}
```
#### Copy
```
{+KC_LGUI}{+KC_C}{-KC_LGUI}8{-KC_LGUI}
```
